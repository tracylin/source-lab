# Supply Chain Aesthetics — Reading Lab

A personal reading companion for developing a critical position on supply chain aesthetics. Curated readings, embedded videos, and a built-in notebook for capturing ideas as essay material.

## Deploy to Vercel

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com), import the repo
3. Vercel auto-detects Vite — click Deploy
4. Done

## Run locally

```bash
npm install
npm run dev
```

## How it works

- **Home**: See all thinkers, track progress
- **Reading**: Embedded articles (iframe) and YouTube lectures with notes alongside
- **Notebook**: All your notes chronologically, tagged by thinker
- **Position**: Your evolving statement on supply chain aesthetics

Data saves to localStorage — persists on the same browser/device.
